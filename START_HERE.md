## 진행 방식

Codex가 알아서 합니다.

- 문제 파일을 직접 열 필요 없음
- JSON, JSONL, SQLite를 알 필요 없음
- 설치나 빌드 없이 대화로 풀이 가능
- 기본 출제는 순차가 아니라 랜덤/균형 출제
- 정답은 답변 후 공개
- 틀리면 핵심 암기 포인트와 올바른 설명 제공

## 권장 답변 방식

O/X 문제:

```text
X. 고구려는 동맹, 동예가 무천.
```

객관식 문제:

```text
3번
```

모르면:

```text
모르겠어. 해설 보여줘.
```

```text
패스
```

## 자연어로 가능한 말

Codex는 아래 표현을 알아듣도록 설정되어 있습니다.

- 시작: `퀴즈 시작`, `문제 내줘`, `한국사 해보자`, `랜덤 문제`, `골고루 내줘`
- 범위: `전근대`, `근현대`, `고려`, `조선`, `현대사`, `파이널1`
- 유형: `객관식`, `O/X`, `순서`, `빈칸`, `단답`, `수업자료`, `키워드 카드`
- 진행: `힌트`, `모르겠어`, `패스`, `해설`
- 복습: `오답`, `틀린 것`, `다시 내줘`, `약한 부분`

## 주의

`draft`는 아직 출제 승인 전 상태입니다. `source_extracted`는 제공된 수업자료에서 확실히 추출했다는 뜻이고, `needs_evidence`는 정답 판단을 별도로 검증해야 한다는 뜻입니다. 평소 채점형 학습에는 `approved` 문제만 사용합니다. 현재 파이널1 O/X 120문항, 근현대사 객관식 5문항, 근현대사 수업자료 기반 빈칸 33문항, 전근대사 객관식/순서 배열 36문항, 전근대사 수업자료 기반 빈칸 23문항은 승인된 채점형 문항입니다. 파이널2 키워드 카드는 승인된 복습 카드이지만 채점형 문제는 아니므로 암기 목록/빈출표현 복습용으로 사용합니다.

이미지나 사진 자체를 보고 답을 고르는 문제는 현재 기본 학습에서 제외합니다. 손글씨나 화면 안 텍스트처럼 읽을 수 있는 자료는 텍스트 학습 재료로 우선 정리합니다.
