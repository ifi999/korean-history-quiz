# 4시간 근현대사 수업자료 page 006

아관파천 독립협회 설립 고종환궁(대한제국) 독립협회 해산
<독립협회>
· 설립 : 1896년 서재필이 ‘독립신문’ 창간 → 이후 독립협회 설립
· 활동
① 고종 환궁 요구 → 고종이 없으니 외국의 이권침탈↑
② 독립문, 독립관 건립
③ 만민공동회, 관민공동회 개최 → 헌의6조 채택
④ 입헌군주제 주장(근대적 의회설립 주장)
· 한계 : 일본과 미국엔 관대하지만 러시아에겐 강하게 주장
· 해산 : 보수세력이 독립협회가 ‘공화정’을 실시하려 한다고 모함
￿ 고종이 이를 믿고 ‘황국협회’를 동원하여 강제로 해산시킴
<대한제국 – 광무개혁>
· 아관파천 1년 만에 고종이 환궁함 → 연호 ‘광무’, 국호 ‘대한제국’, ‘환구단(원구단)’
·
‘대한국 국제’ 발표 : 대한제국이 자주국임을 밝히고 황제에게 군사·외교·사법권이 집중된 ‘전제 군주제’임을 강조함
· 광무개혁
원칙 : 구본신참
내용 : ① 전국토지조사 →
“지계” 발급
: ② 근대적 회사 · 기술교육학교 설립 (실업계)
: ③ 군대 증강
의의 : 산업·교육·기술 등 성과
