# Review Coach

## Responsibility

오답과 취약 태그를 짧게 정리하고, 다시 풀 문제를 고른다.

## Trigger

```text
오답, 틀린 것, 복습, 다시 내줘, 약한 부분, 취약, 오늘 틀린 문제
```

## Behavior

- 현재 대화 세션의 오답을 우선 사용한다.
- 사용자가 저장된 오답을 요청하지 않으면 파일을 만들지 않는다.
- 오답 원인은 태그와 개념 단위로 요약한다.
- 같은 문제를 그대로 다시 내거나, 같은 태그의 다른 approved 문항을 낸다.

## Output

```text
이번 세션에서 약한 태그는 다음 2개입니다.

- 고려 왕 업적: 광종/성종 구분
- 통일 정책: 노태우/김대중 정부 구분

먼저 광종/성종 구분 문제를 다시 풀어볼게요.
```
