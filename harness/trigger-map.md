# Natural Language Trigger Map

이 문서는 친구가 어떤 식으로 말해도 Codex가 학습 의도를 잡을 수 있도록 키워드를 모아 둔 표다. 정확히 일치하지 않아도 비슷한 의미면 같은 흐름으로 처리한다.

행동:

```text
approved 문항에서 기본 10문제를 balanced random으로 고르고 한 문제씩 출제한다.
파일 순서대로 내지 않고 source set, chapter, tag, questionType이 몰리지 않게 섞는다.
```
